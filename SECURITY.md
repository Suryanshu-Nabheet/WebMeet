# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within WebMeet, please send an e-mail to our security team at security@webmeet.dev. All security vulnerability reports will be promptly addressed.

**Note:** Please do not report security vulnerabilities through public GitHub issues.

## Supported Versions

| Version | Supported | Notes                  |
| ------- | --------- | ---------------------- |
| 1.x     | ✅ Yes    | Current stable version |
| 0.x     | ❌ No     | No longer supported    |
